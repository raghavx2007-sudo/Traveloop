## рҹ“Ў API Request/Response Flow

```
Frontend Component
       в”Ӯ
       в”Ӯ 1. User action (e.g., click login)
       в–ј
   useState

       в”Ӯ
       в”Ӯ 2. Call API function
       в–ј
   AuthContext
   login(email, password)
       в”Ӯ
       в”Ӯ 3. Make HTTP request
       в–ј
   Axios Instance
   POST /api/auth/login
   Headers: { Content-Type: application/json }
   Body: { email, password }
       в”Ӯ
       в”Ӯ 4. Request Interceptor
       в”Ӯ    Add Authorization header if token exists
       в–ј
   Network Request
   вҶ’ Backend Server
       в”Ӯ
       в”Ӯ 5. Express Middleware Stack
       в”Ӯ    CORS вҶ’ JSON Parser вҶ’ Routes
       в–ј
   Route Handler
   /api/auth/login
       в”Ӯ
       в”Ӯ 6. Validation Middleware
       в”Ӯ    Check email format, password length
       в–ј
   Controller
   authController.login()
       в”Ӯ
       в”Ӯ 7. Business Logic
       в”Ӯ    вҖў Query database
       в”Ӯ    вҖў Verify password
       в”Ӯ    вҖў Generate JWT
       в–ј
   Database Query
   SELECT * FROM users WHERE email = ?
       в”Ӯ
       в”Ӯ 8. Return result
       в–ј
   Controller Response
   res.json({ success, data: { user, token } })
       в”Ӯ
       в”Ӯ 9. Response Interceptor
       в”Ӯ    Check for 401 errors
       в–ј
   Axios Response
   response.data
       в”Ӯ
       в”Ӯ 10. Update state
       в–ј
   AuthContext
   вҖў setUser(userData)

       в”Ӯ
       в”Ӯ 11. Update UI
       в–ј
   Component Re-render
   setLoading(false)
   Show success message
   Redirect to dashboard
```
