# Traveloop - Travel Planning Platform

A modern, full-stack travel planning application built with React, Node.js, Express, and MySQL.

## Features

### ✅ Completed (Phase 1 & 2)
- 🔐 JWT Authentication (register, login, protected routes)
- ✈️ Trip Management (create, edit, delete trips)
- 📸 Cover image uploads
- 📍 Destination management (add, remove destinations)
- 📊 Dashboard with real-time statistics
- 🔍 Search and sort trips
- 📱 Fully responsive design

### 🚧 Coming Soon (Phase 3 & 4)
- 🎯 Activity management
- 💰 Budget tracking
- ✅ Packing checklist
- 📝 Travel notes/journal
- 🔗 Public trip sharing
- 📈 Advanced analytics

## Tech Stack

**Frontend:**
- React.js with Vite
- Tailwind CSS
- React Router DOM
- Axios
- Context API

**Backend:**
- Node.js
- Express.js
- JWT Authentication
- MySQL

## Project Structure

```
traveloop/
├── client/          # React frontend
└── server/          # Express backend
```

## Getting Started

### Prerequisites
- Node.js (v16+)
- MySQL (v8+)
- npm or yarn

### Installation

1. Clone the repository
2. Set up the database (see database/schema.sql)
3. Install and run backend
4. Install and run frontend

See individual README files in client/ and server/ directories for detailed instructions.
